# Contributing to SmartCalculator

Thanks for your interest in contributing! Please follow these steps to make the process smooth:

1. Fork the repository (or create a branch on the main repo if you have push access).
2. Create a feature branch:
   git checkout -b feature/your-feature
3. Make small, focused changes and include tests where applicable.
4. Commit with clear messages:
   git commit -m "Short summary: detailed explanation"
5. Push your branch and create a pull request. Describe the motivation and changes.

Coding guidelines
- Use consistent formatting (follow existing style). 
- Document public functions and classes.

Reporting bugs
- Open an issue with steps to reproduce and, if possible, a small example.

Thank you for contributing!
